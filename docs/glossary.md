# Glossary

A place to define key terms used in SIKEY.

- **SIK** – Sovereign Identity Key; the root key material owned by a subject.
- **Subject** – The person or entity whose identity is being represented.
- **Issuer** – Entity that issues verifiable credentials / attestations.
- **Verifier** – Entity that verifies proofs or credentials.
- **DID** – Decentralised Identifier.
- **VC** – Verifiable Credential.
- **ZK Proof** – Zero-knowledge proof; a way to prove a statement is true without revealing the underlying data.
