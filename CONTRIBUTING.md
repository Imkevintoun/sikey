# Contributing to SIKEY

Thanks for considering contributing to **SIKEY – Sovereign Identity Key**.

We’re early. Things will change fast. Please be kind, curious, and security-minded.

## Ways to Contribute

- **Ideas & discussion** – open an issue with the `discussion` label.
- **Specs & docs** – improve the manifesto, whitepaper, and specs in `/docs` and `/specs`.
- **Code** – once reference implementations start, help build clients, libraries, or tools.
